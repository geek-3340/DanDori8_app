# 1. アプリの目的
- 道路工事工程管理アプリ
- 施工管理職のユーザーが、現場単位の工程と全現場のスケジュールをガントチャートで管理でき、それに伴うタスクを１日単位で管理できるアプリ。

# 2. ユーザー要件
個人ユーザー（工事現場ごとの工程と不随するタスク管理）

# 3. 機能要件

## 3.1 ユーザー認証機能
- サインアップ／ログイン（ユーザー名＋メール＋パスワード）
- ユーザー名変更
- パスワードリセット

## 3.2 工程管理（MVP）
- 工程登録（再利用できる工程を登録可能）
- 現場の作成/編集/削除
- 現場ごとに工程の登録/削除（工程登録したものから選択or直接入力）
- 現場ごとにガントチャート(3ヶ月)で工程管理
- ガントチャートのカラムは工程単位

## 3.3 全体スケジュール管理（追加機能：初期は実装無）
- ガントチャート(1ヶ月)でスケジュール管理

- ガントチャートのカラムは現場単位
- スケジュールには工程を表示

## 3.4 タスク管理（追加機能：初期は実装無）
- ToDoリスト形式(一日単位)
- リストアイテムの手動作成も可能

# 4. 非機能要件
- レスポンシブ対応（PC／タブレット／スマホ）
- セキュリティ：JWTによる認証、CSRF・XSS対策
- アクセシビリティ：キーボード操作対応

# 5. 主要な画面構成・一部
- トップページ
- ログイン画面／新規登録画面
- ダッシュボード（全体スケジュール）
- 工程管理画面(全体スケジュールの現場をクリックして遷移)
- タスク管理画面

# 6. 技術スタック
- PHP/Laravel
- TypeScript/React
