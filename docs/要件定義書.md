# 1. アプリの目的
- 道路工事工程管理アプリ
- 施工管理職のユーザーが、現場単位の工程と全現場のスケジュールをガントチャートで管理でき、それに伴うタスクを１日単位で管理できるアプリ。

# 2. ユーザー要件
個人ユーザー（工事現場ごとの工程と不随するタスク管理）

# 3. 機能要件

## 3.1 ユーザー認証機能
- サインアップ／ログイン（メール＋パスワード）
- ゲストログイン
- パスワードリセット

## 3.2 工事工程管理
- 工程登録（再利用できる工程を登録可能）
- 現場の作成/編集/削除
- 現場ごとに工程の登録/削除（）
- ボードごとに背景色や画像変更
- ボードの共有（チーム招待）

## 3.3 全体スケジュール管理
- リスト（カラム）の作成／並び替え／削除
- リストタイトルの編集

## 3.4 タスク管理
- カード（タスク）の作成／編集／削除
- カードのドラッグ＆ドロップ移動（リスト間・リスト内）
- 締切・説明の設定
- コメント機能（チーム用）

# 4. 非機能要件
- レスポンシブ対応（PC／タブレット／スマホ）
- パフォーマンス：数百件規模のタスクでも快適に動作
- セキュリティ：JWTによる認証、CSRF・XSS対策
- アクセシビリティ：キーボード操作やスクリーンリーダー対応（できれば）

# 5. 主要な画面構成・一部
- ログイン画面／新規登録画面
- ダッシュボード（ボード一覧）
- ボード画面
- 設定画面

# 6. 技術スタック
- PHP/Laravel
- TypeScript/React
- PostgreSQL
- AWS
- Terraform
