# 1. アプリの目的
- ToDo管理アプリ
- ユーザーがタスクを「ボード」「リスト」「カード」の形式で視覚的に管理・共有できるWebアプリを提供する。プロジェクト管理や日常的なタスク整理の効率化を図る。

# 2. ユーザー要件
個人ユーザー（自分用のタスク管理）

# 3. 機能要件

## 3.1 ユーザー認証機能
- サインアップ／ログイン（メール＋パスワード）
- ゲストログイン
- パスワードリセット
- ソーシャルログイン（将来的に）

## 3.2 ボード管理
- ボードの作成／編集／削除
- ボードごとに背景色や画像変更
- ボードの共有（チーム招待）

## 3.3 リスト管理
- リスト（カラム）の作成／並び替え／削除
- リストタイトルの編集

## 3.4 カード管理
- カード（タスク）の作成／編集／削除
- カードのドラッグ＆ドロップ移動（リスト間・リスト内）
- 締切・説明の設定
- コメント機能（チーム用）

# 4. 非機能要件
- レスポンシブ対応（PC／タブレット／スマホ）
- パフォーマンス：数百件規模のタスクでも快適に動作
- セキュリティ：JWTによる認証、CSRF・XSS対策
- アクセシビリティ：キーボード操作やスクリーンリーダー対応（できれば）

# 5. 主要な画面構成・一部
- ログイン画面／新規登録画面
- ダッシュボード（ボード一覧）
- ボード画面
- 設定画面

# 6. 技術スタック
- PHP/Laravel
- JavaScript
- PostgreSQL
